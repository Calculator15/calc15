# Калькулятор
Приложение «Калькулятор» выполняет стандартные математические действия: сложение/вычитание, умножение/деление, возводит число в степень, находит процент от числа, корни квадратного уравнения, факториал числа, а также вычисляет sin x/arcsin x, cos x/arccos x, tg x/arctg x, ctg x/arcctg x.
